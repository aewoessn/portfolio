This toolbox includes functions that are commonly used within our lab. This documentation has been made very easy to maintain, so please refer to the readme file in the html folder to learn more.

Functions

calcfibang2D Calculates a 2D fiber orientation map
circmean2 Calculates the mean circular direction.
doublewrite Write image with single precision to graphics file
fluorescein_normalization_master Normalizes intensity images based on detector gain and laser power.
freehanddraw Free-hand tracing within an image
loadFlimFitResults Load FLIM results exported from SPCImage
multipageread Load multi-page tiff from graphics file
ndpiread Extracts images from a .ndpi file, outputted by the NanoZoomer.
openSDTFile Loads FLIM decay curve data.
xml2imagedata Extracts header metadata from xml file. Intended for use with Prairie View use only
