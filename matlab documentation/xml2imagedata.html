xml2imagedata

Extracts header metadata from xml file. Intended for use with Prairie View use only

collapse all in page

Syntax

imageData = xml2imagedata(xmlFilename)

Description

imageData = xml2imagedata(xmlFilename) reads the xml file, specified by xmlFilename, and returns a structure, imageData, containing all of the header information about the collected images (i.e. pmt gain, and image resolution). For the X/Y/Z positions, please note that the header metadata contains only the position when "Acquire image" was selected. These values may not accurately represent the position for collected images.

Input Arguments

The absolute or relative path for the XML file where the metadata is intended to be extracted.

Data Types: char | string

All metadata from header of XML file.
